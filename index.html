¿Qué es el desarrollo de videojuegos?

 Es el proceso de diseñar, programar y probar un videojuego . El esfuerzo lo realiza un desarrollador , que va desde una sola persona hasta un equipo internacional diseminado por todo el mundo. 

OBJETIVO

Mostrar el desarrollo de los videojuegos, asi como tambien elaborar estos mismos para el disfrute de las personas y mostrar la cultura que tiene la industria.

Artic Studio

¿Quienes Somos?

Somos una casa desarrolladora de videojuegos 2D con el objetivo de representar el desarrollo de este mismo así como también buscar la manera de darse a conocer para futuros proyectos de mayor ambición.

A continuacion se presentaran los modulos mas destacables en los cuales nos desenvolvemos.

Gestion de Base de Datos

La gestión de datos es el proceso de ingerir, almacenar, organizar, utilizar y mantener los datos creados y recopilados por una organización. La gestión de datos eficaz es una pieza fundamental de la implementación de los sistemas de TI que ejecutan aplicaciones comerciales y proporcionan información analítica para ayudar a impulsar la toma de decisiones operativas 

Marketing

es el proceso de exploración, creación y entrega de valor para satisfacer las necesidades de un mercado objetivo 

Programacion

La programación es el proceso de crear un conjunto de instrucciones que le dicen a una computadora como realizar algún tipo de tarea.

Diseño de Videojuegos

El  diseño de  videojuego s  incluye la creación de historias, personajes, metas, reglas y desafíos que impulsen las interacciones con otros personajes, usuarios u objetos

Mi Perfil:

  • Programador
  • Editor de video y fotos
  • Diseñador de Videojuegos
  • Gestor de bases de datos

Mi objetivo con Artic

Busco explotar mi potencial en los diferentes campos que conlleva el desarrollo de videojuegos y la publicacion de estos mismos, asi como tambien concientizar a las personas sobre el desarrollo de estos y demostrar el significado que tienen en la sociedad.

Premios para la Feria

Para la feria del programador, se presentaran diferentes titulos del genero arcade para el disfrute de los usuarios, y si se llega a la meta de puntos fijada se daran premios.

MAS INFORMACION

¿Qué ofreceremos en la feria?

Se presentaran 3 videojuegos del estilo arcade los cuales tienen sistema de puntaje y estaran disponibles para todos, con el fin de que los jueguen y obtengan su puntaje y con este mismo participar en recompensas segun puntaje

Pong

¿Como funcionara este sistema?
Si se obtiene el puntaje estimado, se les dara un premio standard y asi como tambien se recompensara la competitividad y se les dara premios al primer, segundo y tercer lugar de cada juego.
